# GitHub Pages Configuration

To enable GitHub Pages for your Quarto site:

1. Go to your repository settings on GitHub.
2. Navigate to the "Pages" section.
3. Set the source to the `gh-pages` branch (created by the GitHub Action).
4. Save the settings.

Your site will be published at:
https://<your-github-username>.github.io/<your-repo-name>/

The GitHub Action will automatically render and publish your site whenever you push changes to the main or master branch.
